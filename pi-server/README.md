# LocalLearning Pi Server

The classroom hub. Runs on a Raspberry Pi 5 (8 GB) connected to a USB microphone,
serving live multilingual captions to student phones over local WiFi.

## Stack

- **FastAPI** + uvicorn for HTTP and SSE
- **whisper.cpp** (`tiny.en`) via [pywhispercpp](https://pypi.org/project/pywhispercpp/) — real-time English transcription
- **Gemma 4 E2B** (Q4_K_M) via [llama-cpp-python](https://pypi.org/project/llama-cpp-python/) — streaming translation + study pack generation
- **sounddevice** for cross-platform audio capture (works on Pi and Mac dev box)

## One-time setup on the Pi

```bash
cd pi-server
./scripts/setup-pi.sh
```

The script installs system packages, creates a venv, installs Python deps, and
downloads model weights into `./models/`.

## Running

```bash
source .venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8080
```

The Pi will print its IP. Project a QR code for that URL on the teacher's screen
(automatically generated by the PWA once a class starts).

## Local development on a Mac/Linux laptop

The same code runs without the Pi — useful for iterating on the API and PWA
before testing on real hardware. You'll still need the model files locally
(or stub them out by removing the `WhisperTranscriber` / `GemmaTranslator` calls).

```bash
cd pi-server
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8080
```

## API quick reference

| Method | Path | Purpose |
|--------|------|---------|
| `GET` | `/api/health` | Health + active class id |
| `POST` | `/api/class` | Start a class (`{title, teacher?}`) |
| `POST` | `/api/class/{id}/end` | End a class |
| `GET` | `/api/class/active` | Currently-live class (or `204`) |
| `GET` | `/api/class/{id}` | Class metadata + counts |
| `GET` | `/api/qr/{id}` | PNG QR code that resolves to the join URL |
| `GET` | `/api/stream/{id}/{lang}` | SSE: live captions in `lang` |
| `POST` | `/api/class/{id}/confusion` | Mark a caption as confusing |
| `GET` | `/`, `/join` | Student PWA |

Languages currently wired: `en`, `ar`, `uk`, `es`, `zh`, `fr`, `ps`, `fa`.

## Layout

```
app/
  main.py          FastAPI app + endpoints
  config.py        Settings (env-overridable, prefix LL_)
  models.py        Pydantic data models
  store.py         In-memory store of the active lecture
  transcription.py Audio capture + whisper.cpp loop
  translation.py   Gemma translator + per-language SSE bus
scripts/
  setup-pi.sh      One-time Pi setup
tests/             (TODO)
```
